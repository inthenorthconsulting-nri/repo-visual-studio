# technical-grid

The "contextual wildcard" for technical audiences: a dense, grid-aligned
light theme with monospace-friendly code blocks and a GitHub-flavored blue
accent, built for architecture reviews and design reviews where diagrams and
code carry as much weight as prose.

Best for: `architecture-review` audience profile, design review, code
walkthrough.
